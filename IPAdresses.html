IP Addresses

An IP or Internet Protocol address is an indicator that allows other systems within the network to identify others. In modern day use the two most commonly used fomrats are IPv4 and IPv6 addresses. An IPv4 address consist of four sets of number ranging from 0 to 255. All sets numbers on a IP address are separated by a dot or period. IP addresses are registered through the InterNIC. These particular IP addresses has three different classes; C (smallest), the next largest is B, and the largest is A. The IPv6 format is significantly larger than the IPv4 while it contain eight sets of four digits and uses colons also. An example of an IPv6 address is: 2602:0445:0000:0000:a93e:5ca7:81e2:5f9d.
